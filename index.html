Projects

DInv (automatic distributed invariant detector)

Distributed systems are complex and require maticulous development to correctly handel asynchrounous execution, and partial failures. DInv is a tool for automatically detecting likely data invariants in distributed systems. Data invariants are useful for verifing the correctness of a system

Dovid (automatic distributed system documenter)

Distibuted systems can be diffult to understand from just reading source code. The most integeral part of a distributed system is its network communication. Dovid automatically detects network communication and generates documentation for variables which either affect or are affected by network communication.
